BRAIN GAME

Popularly known as "SIMON GAME"

  • Simon is named after the simple children's game of Simon Says.
  • The game contains four colored buttons, each producing a particular tone when it is clicked or activated by the system.
  • A round in the game consists of the device lighting up one button in each level, after which the player must reproduce the order of buttons from the very starting of the game by clicking the buttons.
  • As the game progresses, the number of buttons to be pressed increases.
  • Level is incremented each time the user is able to reproduce the correct sequence.

Why it's named BRAIN GAME ?

  • The game indirectly tests the memorising power of user's brain.
  • How well the user is at remembering the sequences or pattern?
